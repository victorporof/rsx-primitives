A matrix storage that does not contain any reference and that is stored contiguously in memory.

The storage requirement means that for any value of i in [0, nrows * ncols[, the value .get_unchecked_linear succeeds. This trait is unsafe because failing to comply to this may cause Undefined Behaviors.
